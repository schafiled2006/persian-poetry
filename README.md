# persian-poetry    # گنجینه اشعار پارسی

یه وب‌سایت تعاملی برای نمایش اشعار شاعران بزرگ ایرانی که با HTML، CSS و JavaScript خالص ساخته شده.

---

## معرفی

گنجینه اشعار پارسی یک وب‌اپلیکیشن تک‌صفحه‌ای‌ست که اشعار ۱۱ شاعر بزرگ ایرانی رو با طراحی Glassmorphism و فونت ایران نستعلیق نمایش میده. کل پروژه بدون هیچ فریم‌ورک یا کتابخانه‌ای نوشته شده و فقط از HTML5، CSS3 و Vanilla JavaScript استفاده کرده.

---

## ویژگی‌ها

- نمایش ۲۰ شعر از ۱۱ شاعر مختلف
- فونت ایران نستعلیق برای نمایش اصیل اشعار
- بک‌گراند ثابت با طراحی ایرانی-اسلامی
- افکت ذرات متحرک در پس‌زمینه با Canvas API
- طراحی Glassmorphism یا همون کارت‌های شیشه‌ای
- سیستم علاقه‌مندی‌ها با ذخیره‌سازی در LocalStorage
- کپی کردن شعر با یک کلیک از طریق Clipboard API
- اشتراک‌گذاری شعر با Web Share API
- نمایش حال و هوای هر شعر
- شمارنده شعر که شماره فعلی از کل رو نشون میده
- دکمه‌های ناوبری برای رفتن به شعر بعدی و قبلی
- طراحی کاملا واکنش‌گرا برای موبایل و دسکتاپ
- صفحه لودینگ با انیمیشن
- نوتیفیکیشن برای اطلاع‌رسانی به کاربر

## نصب و اجرا

روش اول - اجرای مستقیم

فایل `index.html` رو مستقیم توی مرورگر باز کن. نیازی به نصب هیچ چیزی نیست.

روش دوم - Live Server در VS Code

```
1. پروژه رو در VS Code باز کن
2. افزونه Live Server رو نصب کن
3. روی index.html راست‌کلیک کن و Open with Live Server رو بزن
```

روش سوم - کلون از GitHub

```bash
git clone https://github.com/username/persian-poetry.git
cd persian-poetry
```

نکته: برای نمایش درست فونت IranNastaliq، فایل `font/IranNastaliq.ttf` باید در مسیر درست قرار داشته باشه.

## ویژگی‌های فنی

```css
/* CSS Variables برای مدیریت رنگ‌ها */
:root {
  --primary-color: ...;
  --glass-bg: rgba(255, 255, 255, 0.1);
}

/* Glassmorphism Effect */
.card {
  backdrop-filter: blur(20px);
  background: var(--glass-bg);
}
```

- CSS Variables برای مدیریت متمرکز رنگ‌ها و تم
- backdrop-filter برای افکت شیشه‌ای
- Canvas API برای رندر ذرات متحرک در پس‌زمینه
- Clipboard API برای کپی کردن شعر
- Web Share API برای اشتراک‌گذاری بومی در موبایل
- LocalStorage برای نگه داشتن علاقه‌مندی‌ها بین session‌ها
- انیمیشن‌های CSS3 برای transition و transform
- رویکرد Mobile-First در طراحی
- کامنت‌گذاری فارسی در کد
